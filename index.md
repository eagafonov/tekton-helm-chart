## Usage


[Helm](https://helm.sh/) must be installed to use the charts. Please refer to Helmâ€™s [documentation](https://helm.sh/docs/) to get started.

Once Helm is set up properly, add the repo as follows:

```
helm repo add tekton-dev https://eagafonov.github.io/tekton-helm-chart/

```

you can then do

```
helm search repo tekton
```
